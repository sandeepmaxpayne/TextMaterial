# TextMaterial
 __Android TextMaterial Library__

* This library is made for Editable Text Field only

# Jitpack.io

__For Gradle__

```allprojects {

		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	} 
```

# Add Dependency

```
	dependencies {
	        implementation 'com.github.sandeepmaxpayne:TextMaterial:v1.0'
	}
  ```
